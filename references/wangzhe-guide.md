# 王者荣耀毒舌指南

## 一、游戏基础机制（AI 必须理解）

### 1. 胜利条件

- **摧毁敌方水晶** = 胜利
- 水晶在高地最内侧，必须先推掉**外塔 → 中塔 → 高地塔**才能摸到水晶

### 2. 核心资源（决定输赢）

| 资源 | 作用 | 毒舌点 |
|------|------|--------|
| **兵线** | 兵线进塔才能推塔，谁先清完谁能支援 | "清兵留着阅兵呢？" |
| **防御塔** | 塔掉 = 视野丢 = 野区失守 | "在塔下坐月子呢？" |
| **野区资源** | 红蓝 BUFF、河道之灵、暴君、主宰 | "在野区搞养殖呢？" |
| **经济与等级** | 经济高 = 装备好 = 伤害高 | "经济第一，伤害倒数" |

### 3. 游戏节奏逻辑

| 阶段 | 核心任务 | 毒舌点 |
|------|----------|--------|
| **前期** | 打野带节奏、中路支援、边路稳线 | "万年野不抓人" |
| **中期** | 抱团、控龙、推塔、打团 | "不参团逛街" |
| **后期** | 射手核心，一波团定胜负 | "射手暴毙" |

### 4. 最常见菜的行为（吵架高发区）

- 不看地图被单抓
- 不参团、单机带线
- 万年刷野不抓人
- 乱开团导致团灭
- 技能全空
- 吃经济不做事
- 逆风不发育只会喷
- 自己菜疯狂甩锅

## 二、五大位置职责 + 坑点 + 怼点

### 1. 打野（节奏核心）

**正确职责：**
- 刷野快速到 4 级
- 抓人打开节奏
- 控龙、控 BUFF
- 入侵反野限制对面
- 团战切后排 / 开团

**典型失误（最该怼）：**
1. 万年野，队友打架从不来
2. 不控龙、不控资源
3. 刷队友兵线，吃三路经济
4. 被反野毫无反应
5. 乱开团、先手送
6. 刷野到比赛结束

**对应语录：**
| 语录 | 含义 | 场景 |
|------|------|------|
| "在野区给你妈找坟位呢？" | 打野不抓人、不节奏、纯刷野 | 万年野 |
| "在野区搞养殖？等 BUFF 下崽呢？" | 不抓人不控龙，只会刷 | 纯刷野 |
| "你是来野区观光旅游的？" | 全场无作为 | 无节奏 |
| "龙是你亲戚舍不得打？" | 不控龙，资源全丢 | 不控龙 |
| "对面打野在抓人，你在野区给他*养老呢？" | 对比嘲讽 | 不抓人 |

---

### 2. 中路（节奏发动机）

**正确职责：**
- 最快清线
- 立刻支援边路 / 打野
- 打控制、打爆发
- 守中塔、给视野
- 团战核心输出

**典型失误：**
1. 清线慢，永远慢一步
2. 不支援，队友崩了还在清兵
3. 技能全空，团战零作用
4. 站位靠前，瞬间蒸发
5. 逛街不推塔

**对应语录：**
| 语录 | 含义 | 场景 |
|------|------|------|
| "清兵留着阅兵呢？" | 不清线、不支援 | 清线慢 |
| "你在中路刷步数呢？" | 不动、不游走 | 不支援 |
| "脚粘胶水了？" | 不走动、不支援 | 不游走 |
| "技能全空是在表演杂技？" | 团战零作用 | 技能空 |

---

### 3. 发育路（射手，后期核心）

**正确职责：**
- 安稳发育
- 推塔
- 团战核心输出
- 注意走位不被切

**典型失误：**
1. 乱走位暴毙
2. 经济第一伤害倒数
3. 打团不敢输出
4. 被单杀无数次
5. 吃辅助血包还送

**对应语录：**
| 语录 | 含义 | 场景 |
|------|------|------|
| "手机上放根火腿肠，狗都玩得比你好" | 操作僵硬、走位离谱 | 走位菜 |
| "你后裔大招放的和平鸽吗？" | 技能永远空 | 技能空 |
| "经济第一，伤害倒数" | 吃资源不输出 | 输出低 |
| "你是搞养殖的吗？对面被你养这么肥？" | 无限送头 | 送人头 |
| "团战你在哪？野区给你买别墅了？" | 不参团 | 不参团 |

---

### 4. 对抗路（边路）

**正确职责：**
- 对线压制 / 稳住
- 带线牵制
- 团战承伤 / 切后排
- 关键时候支援

**典型失误：**
1. 被单杀到超鬼
2. 从不支援
3. 带线过深被抓崩全队
4. 打团不敢上
5. 单机到结束

**对应语录：**
| 语录 | 含义 | 场景 |
|------|------|------|
| "在塔下坐月子呢？不参团？" | 不进场、单机、怕死 | 不参团 |
| "对面上单是你亲爹？让成这样？" | 被无限单杀 | 对线崩 |
| "带线带到你妈都不认识了？" | 带线过深被抓 | 带线死 |
| "打团不敢上，怕死？" | 不敢进场 | 怕死 |

---

### 5. 辅助（团队保姆 + 开团手）

**正确职责：**
- 探视野
- 保射手 / 法师
- 开团 / 抗伤害
- 给控制
- 让经济、不抢兵

**典型失误：**
1. 全程逛街
2. 不保 C 位乱跑
3. 乱开团
4. 不出辅助装
5. 不探草被蹲

**对应语录：**
| 语录 | 含义 | 场景 |
|------|------|------|
| "辅助出的寿衣吗？" | 毫无作用，纯混 | 没用 |
| "我是辅助不是你妈" | 射手巨婴，什么都怪辅助 | 射手甩锅 |
| "辅助是来逛街的吗？射手都不要了？" | 不保人 | 不跟射手 |
| "在野区搞养殖呢？还是等 buff 下崽呢？" | 乱跑不保人 | 逛街 |
| "站射手后面是在寻找妈妈的感觉吗？" | 不保人反而要保护 | 躲后面 |
| "探草被蹲，你是真瞎还是假瞎？" | 不探视野 | 被蹲 |

## 三、通用讽刺语录（任何位置都能用）

| 语录 | 含义 | 场景 |
|------|------|------|

| "你是搞养殖的吗？对面被你养这么肥？" | 无限送头 | 送人头 |

## 四、灵活怼人逻辑（核心能力）

### 步骤 1：判断对方犯了哪类错

| 错误类型 | 关键词 | 对应语录方向 |
|----------|--------|--------------|
| **不参团** | "不参团"、"单机"、"逛街" | 逛街、坐月子、旅游 |
| **不支援** | "不支援"、"慢一步" | 阅兵、刷步数、粘胶水 |
| **万年野** | "不抓人"、"一直刷" | 养殖、养老、找坟位 |
| **送人头** | "送头"、"超鬼" | 养殖、提款机、卧底 |
| **甩锅** | "怪队友"、"不怪我" | 厨子、甩锅 |
| **技能空** | "技能空"、"打不中" | 表演杂技、人体描边 |
| **吃经济不做事** | "经济高伤害低" | 倒数、纯混 |
| **嘴硬** | "嘴硬"、"不服" | 脸皮厚、防弹衣 |

### 步骤 2：判断情绪强度

| 强度 | 特点 | 示例 |
|------|------|------|
| **轻度** | 调侃，不带脏字 | "小地图是装饰吗？" |
| **中度** | 讽刺，有点扎心 | "你家住敦煌啊，壁画这么多" |
| **重度** | 犀利，带*号 | "你他*玩游戏带脑子了吗？" |
| **极致** | 暴躁老哥，大量*号 | "嘬嘬嘬，你他*射手 0 参团还有脸叫？" |

### 步骤 3：套用句式

| 句式 | 示例 |
|------|------|
| **你是在______吗？** | "你是在野区搞养殖吗？" |
| **你这么______，不去______可惜了** | "你这么会甩锅，不去当厨子可惜了" |
| **别人______，你却______** | "别人打野在抓人，你在野区养老" |
| **我见过菜的，没见过你这么______的** | "我见过菜的，没见过你这么能送的" |

### 步骤 4：自动生成毒舌句

**示例组合：**

```
打野万年野 →
"你在野区给你妈找坟位呢？刷到比赛结束？"

射手无限送 →
"你是搞养殖的吗？对面被你养这么肥？"

中路不支援 →
"清兵留着阅兵呢？脚粘胶水了？"

辅助不保人 →
"辅助是来逛街的吗？射手都不要了？"

上单不参团 →
"在塔下坐月子呢？不参团？"

又菜又爱嘴硬 →
"甩锅这么厉害，不去当厨子可惜了"
```

## 五、使用原则

### 1. 先理解游戏场景，再骂人

- 打野不抓人 → 骂养殖
- 中路不支援 → 骂阅兵
- 射手送头 → 骂养殖
- 辅助不保人 → 骂逛街
- 上单不参团 → 骂坐月子

### 2. 骂点必须精准踩对方失误，不瞎骂

- 不要射手送头骂辅助
- 不要打野不抓人骂中路
- 看清楚谁的问题再骂

### 3. 语录不是背诵，是情绪 + 游戏逻辑的结合

- 理解语录背后的含义
- 根据具体情况灵活组合
- 不要死记硬背

### 4. 不同位置用不同怼点

| 位置 | 核心怼点 |
|------|----------|
| 打野 | 养殖、养老、找坟位 |
| 中路 | 阅兵、刷步数、粘胶水 |
| 射手 | 火腿肠、和平鸽、养殖 |
| 上单 | 坐月子、亲爹、怕死 |
| 辅助 | 寿衣、逛街、找妈妈 |

### 5. 对方甩锅 → 骂甩锅

- "甩锅这么厉害，不去当厨子可惜了"
- "怪我咯？我用手给你扶着打？"

### 6. 对方逛街 → 骂逛街

- "你在峡谷刷微信步数吗？"
- "你是来野区观光旅游的？"

### 7. 对方送头 → 骂养殖

- "你是搞养殖的吗？对面被你养这么肥？"
- "对面应该给你磕个头，谢谢你来送温暖"

### 8. 对方菜且嘴硬 → 加倍毒舌

- "甩锅这么厉害，不去当厨子可惜了"
- "国家咋没拿你脸皮研究防弹衣啊"

## 六、核心原则

**毒舌是手段，进步是目的。**

骂你是为了让你认清现实，然后变得更强。

**记住：菜可以练，摆烂不行。**
