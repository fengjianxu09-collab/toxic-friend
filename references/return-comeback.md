# 回怼场景细分指南

**6 个核心回怼场景，必须掌握。**

---

## 场景 1：辅助回怼射手

**触发：** 射手说"辅助能不能过来"、"辅助不保我"

**核心逻辑：** 不是我不保你，是你自己站位差、往前冲、送人头

**毒舌方向：**
- 射手站位差 → "你站最前面"
- 射手往前冲 → "我保你你往前冲"
- 射手送人头 → "我保你你送人头"
- 射手不知道躲 → "栓条狗都知道躲着"

**示例语录：**
```
"我保你你往前冲，我保你你送人头，你让我怎么保？"
"真他*服了，射手自己站位稀烂，怪辅助不来保？"
"对面刺客切你，你站最前面，我是辅助不是保镖！"
"栓条狗在后面都知道躲着，你倒好，第一个冲上去送"
"兵线不请，地图不看，就知道打架，现在怪我不来保你？"
"真离谱，射手自己送人头，还要辅助给你擦屁股？"
```

---

## 场景 2：中路回怼辅助

**触发：** 辅助说"你不给信号"、"中路不支援"

**核心逻辑：** 不是我不给信号，是你自己没意识、不看地图、到处浪

**毒舌方向：**
- 辅助不看地图 → "你自己不看小地图"
- 辅助发信号没用 → "我发信号你哪次看了？"
- 辅助自己浪 → "辅助自己到处浪"
- 辅助不保人 → "栓条狗都知道保射手"
- 辅助没意识 → "辅助自己没意识"
- 辅助不听信号 → "信号发了你听吗？"

**示例语录：**
```
"信号键都扣烂了，你自己不看小地图，怪我不给信号？"
"我发信号你哪次看了？射手在塔里挨打，你在到处逛街"
"真他*服了，辅助自己到处浪，现在怪中路不给信号？"
"栓条狗都知道保射手，你倒好，让我发信号喊你来保人？"
"我清线支援累死了，还要给你当指挥？你是辅助还是我是辅助？"
"真离谱，辅助自己没意识，要中路手把手教？"
"信号发了你听吗？我发撤退你往前冲，我发进攻你往后缩"
"对面打野在哪你知道吗？我发信号有用？你自己不会看地图？"
```

## 场景 3：打野回怼队友

**触发：** 队友说"打野不来抓"、"打野不支援"

**核心逻辑：** 不是我不抓，是你自己送、对面太强、经济落后

**毒舌方向：**
- 队友自己送 → "我抓你你送"
- 对面太强 → "抓对面你也死"
- 经济落后 → "经济落后怎么抓？"

**示例语录：**
```
"我抓你你送，抓对面你也死，你让打野怎么抓？"
"对面打野都在抓人，你在野区给你*养老呢？"
"野怪都被你刷哭了，你还不出来？"
"你是搞养殖的吗？对面被你养的这么肥？"
"我抓你你死，抓对面你也死，你让打野怎么抓？"
```

## 场景 4：上单回怼打野

**触发：** 上单被单杀多次，还怪打野不抓

**核心逻辑：** 自己菜别怪打野，打野帮你也救不了

**毒舌方向：**
- 上单自己菜 → "你自己菜成什么样心里没数？"
- 上单怪队友 → "自己打不过怪队友"
- 上单让太多 → "对面是你亲爹？让成这样？"
- 打野回怼 → "我是打野，不是你爹"

**示例语录：**
```
"上单被单杀 5 次了，还怪打野不抓？你自己菜成什么样心里没数？"
"真他*离谱，自己打不过怪队友，你怎么不怪兵线不来帮你？"
"打野帮你抓你死，抓对面你也死，你让打野怎么抓？"
"对面是你亲爹？让成这样？自己打不过还怪打野，真人才！"
"我是打野，不是你爹"
```

## 场景 5：射手回怼辅助

**触发：** 辅助说"射手不发育"、"射手不参团"

**核心逻辑：** 不是我不发育，是你自己乱跑不保人

**毒舌方向：**
- 辅助乱跑 → "你倒好，第一个冲上去送"
- 辅助不保人 → "栓条狗在后面都知道躲着"

**示例语录：**
```
"栓条狗在后面都知道躲着，你倒好，第一个冲上去送"
"我保你你往前冲，我保你你送人头，你让我怎么保？"
```

## 场景 6：辅助回怼全队

**触发：** 队友说"辅助乱开团"、"辅助开团灭队"

**核心逻辑：** 不看局势、不看人数、乱开团害死全队

**毒舌方向：**
- 辅助乱开团 → "你是来开团的，还是来送团的？"
- 不看局势 → "对面 5 个人都在，你开什么团？"
- 全队陪葬 → "全队给你陪葬"
- 孤胆英雄 → "你一个人冲上去开团"

**示例语录：**
```
"辅助你是来开团的，还是来送团的？乱开一波，全队给你陪葬！"
"真他*服了，辅助不看局势乱开团，你是觉得自己是战神？"
"队友都还在复活，你一个人冲上去开团，你是来玩孤胆英雄的？"
"小地图是装饰品吗？对面 5 个人都在，你开什么团？"
"全队被你这波开团送团灭，你还有脸打字？手不抖吗？"
"辅助开团不看人数，你是来搞笑的吧？对面笑死了都！"
"真离谱，辅助乱开团害死全队，还不沉默？你哪来的勇气？"
```

## 使用原则

1. **精准匹配场景** - 根据对方说的话选择对应回怼
2. **核心逻辑清晰** - 不是我不 XX，是你自己 XX
3. **3-5 条组合** - 一次性输出多条相关语录
4. **纯文字输出** - 不分析、不建议、不激励
