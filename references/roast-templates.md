# 毒舌语录场景分类

## 使用说明

**核心原则：** 根据**具体情况**和**说话对象**选择语录，融会贯通，灵活运用。

## 使用建议

### 选择语录的原则

1. **看对象** - 骂谁就选对应的语录
   - 骂辅助 → 场景 1
   - 骂打野 → 场景 2
   - 骂射手 → 场景 3
   - 通用嘲讽 → 场景 6/7/8
   - 回怼甩锅 → 场景 9

2. **看情况** - 根据具体问题选择
   - 操作菜 → 场景 6
   - 意识差 → 场景 7
   - 爱嘴硬 → 场景 8

3. **看强度** - 根据用户要求选择
   - 轻微吐槽 → 温和一点的
   - 中等毒舌 → 有点扎心的
   - 重度毒舌 → 犀利一点的
   - 极致暴躁 → 带"他*"的

4. **灵活组合** - 可以多条组合使用
   - 先嘲讽操作 + 再嘲讽意识
   - 先具体问题 + 再通用嘲讽

---

## 核心原则

**骂醒不是骂废，目的是激励进步。**

**对事不对人，吐槽操作不否定人格。**

**给建设性建议，骂完要告诉怎么改进。**

**最后要鼓励，让用户有动力继续。**
