## 🎯 核心特点

| 特点 | 说明 |
|------|------|
| **精准踩点** | 根据位置和错误类型选择毒舌方向 |
| **基于模式创新** | 理解语录模式，创造新内容，不机械搬运 |
| **深入游戏机制** | 结合兵线、塔、野区、经济、龙、BUFF |
| **纯文字输出** | 不分析、不建议、不激励，3-5 条组合 |
| **用*号替代** | 敏感词用*号替代（死→*、妈→*） |

## 📚 语录场景分类

### 骂队友场景

| 场景 | 毒舌方向 | 示例 |
|------|----------|------|
| 辅助不跟射手 | 逛街、找妈妈、寿衣 | "射手在塔里挨打，你在到处逛街" |
| 打野不抓人 | 养殖、养老、找坟位 | "你在野区给你*养老呢？" |
| 射手不参团 | 冲锋、演员、养殖 | "射手你是来玩冲锋陷阵的？" |
| 上单不带线 | 坐月子、亲爹 | "你还在塔下坐月子呢？" |
| 中单不支援 | 阅兵、刷步数 | "清完线就发呆，是在等中彩票？" |

### 回怼场景

| 回怼场景 | 核心逻辑 | 示例 |
|----------|----------|------|
| 辅助回怼射手 | 不是我不保你，是你自己站位差 | "我保你你往前冲，我保你你送人头" |
| 中路回怼辅助 | 不是我不给信号，是你自己没意识 | "你自己不看小地图，怪我不给信号？" |
| 打野回怼队友 | 不是我不抓，是你自己送 | "我抓你你送，抓对面你也死" |
| 上单回怼打野 | 自己菜别怪打野 | "你自己菜成什么样心里没数？" |

## 🔧 技术细节

- **位置识别**：自动提取位置（辅助/打野/射手/上单/中单）
- **问题识别**：自动识别错误（不跟/不抓人/不参团/送头/甩锅）
- **意图判断**：判断用户想要（骂队友/骂自己/回怼）
- **语录生成**：基于模式创新，创造新内容
- **敏感词处理**：用*号替代敏感词
