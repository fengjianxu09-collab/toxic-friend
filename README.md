你被队友气到血压升高？
你截图发给朋友问"这操作是人能打出来的"？
你盯着 0-5 的战绩想摔手机？
你想骂人但怕被举报禁言？
你看着队友送人头却无能为力？

**把被坑的愤怒交给 AI，发送前先骂爽一遍。**

提供队友的坑爹操作、社交截图、你的描述
生成一个精准的毒舌回怼
预测队友怎么回你，或者直接帮你骂回去

---

⚠️ 本 skill 仅用于个人娱乐吐槽，不用于人身攻击、网络暴力或骚扰他人。游戏而已，开心最重要。

## 🚀 使用

直接对 AI 说：

- "中路不支援，怎么怼"
- "打野不抓人，怎么骂"
- "射手不参团，怎么骂"
- "辅助不跟射手，怎么骂"
- "我是辅助，射手说我不保人，怎么回怼"

## 💬 效果示例

### 射手往前冲送人头

> "射手你是来玩冲锋陷阵的？不知道自己是脆皮？"
> "栓条狗在后面都知道躲着，你倒好，第一个冲上去送"
> "对面刺客笑死了，还有射手自己送上门的"

### 辅助不保人

> "射手在塔里挨打，你在到处逛街，你是来旅游的？"
> "信号键都扣烂了，支援一下会*啊？"
> "栓条狗在射手旁边都不会打成这样"

### 打野不抓人

> "对面打野都在抓人，你在野区给你*养老呢？"
> "野怪都被你刷哭了，你还不出来？"
> "你是搞养殖的吗？对面被你养的这么肥？"

## 🎯 核心特点

- **精准踩点** - 根据位置和错误类型选择毒舌方向
- **基于模式创新** - 理解语录模式，创造新内容
- **深入游戏机制** - 结合兵线、塔、野区、经济、龙、BUFF
- **纯文字输出** - 不分析、不建议、不激励，3-5 条组合
- **用*号替代** - 敏感词用*号替代（死→*、妈→*）
