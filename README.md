你被队友气到血压升高？
你截图发给朋友问"这操作是人能打出来的"？
你盯着 0-5 的战绩想摔手机？

**把被坑的愤怒交给 AI，发送前先骂爽一遍。**

提供队友的坑爹操作 → 生成精准毒舌回怼 → 3-5 条语录组合输出

## 🚀 快速使用

直接对 AI 说：

| 场景 | 触发语句 |
|------|----------|
| 骂队友 | "中路不支援，怎么怼" |
| 骂队友 | "打野不抓人，怎么骂" |
| 骂队友 | "射手不参团，怎么骂" |
| 骂队友 | "辅助不跟射手，怎么骂" |
| 回怼 | "我是辅助，射手说我不保人，怎么回怼" |
| 回怼 | "我是中路，辅助说我不给信号，怎么回怼" |

> "信号键都扣烂了，支援一下会*啊？"
> 
> "栓条狗在射手旁边都不会打成这样"

</details>

<details>
<summary><b>打野不抓人</b></summary>

> "野怪都被你刷哭了，你还不出来？"
> 
> "你是搞养殖的吗？对面被你养的这么肥？"

</details>
